PENDAHULUAN

Curriculum Vitae (CV) adalah daftar riwayat hidup yang berisi data dan informasi yang diperlukan antara lain untuk melamar pekerjaan. Biasanya CV disajikan di aplikasi MS Word, MS Power Point atau aplikasi desain grafis lainnya. Di sini penyaji coba menyajikan CV dalam berbasis HTML yang dapat diakses dengan menggunakan peramban internet.

Apa itu HTML ? Hypertext Markup Languange atau yang disingkat HTML adalah bahasa standar pemograman yang digunakan untuk membuat halaman website. HTML adalah fondasi dari sebuah website, sedangkan untuk mempercantik tampilannya kita dapat menggunakan CSS. CSS atau Cascading Style Sheet itu sendiri adalah salah satu styling languange atau bahasa desain, bagian dari markup languange yang digunakan untuk mendesain suatu halaman website

Mengapa penyaji memilih menggunakan HTMl dan CSS dalam membuat curriculum vitae ? penyaji berpendapat bahwa bahasa ini yang paling mudah digunakan dan ringan untuk hampir semua gadget. Tidak diperlukan aplikasi berbayar untuk menggunakan HTML dan CSS. Dan media yang digunakan untuk membuat dan menampilkan CV berbasi HTML ini pasti dapat ditemukan di hampir semua perangkat yang pembaca gunakan.
